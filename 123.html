图1：方式一 — LF-Tag 绑定到 IAM Role，用户 AssumeRole 后获得权限

方式一：LF-Tag 授权绑定到 IAM Role 上。用户通过 AssumeRole 代入 IAM Role，Lake Formation 用 Role 上的 LF-Tag 匹配表的 LF-Tag。
完整链路：用户身份（Assume IAM Role）→ Athena 执行（Athena Service Role）→ LF 读数据（DataAdmin Service Role）。

图2：方式二 — LF-Tag 绑定到 Session Tag，用户通过 SAML AssumeRole 时传递

方式二：LF-Tag 授权绑定到 Session Tag 上。用户通过 AD 认证登录，AD Profile 属性映射为 Session Tag，
通过 SAML 2.0 AssumeRoleWithSAML 传递 Session Tag，Lake Formation 用 Session Tag 匹配表的 LF-Tag。
